# Desktoper EN
Program in python that is used to have more virtual like desktops with other icons on each

# How to set up
1. Make some folders and name them (in file Plochy.dsk is some inspiration).
2. Write the names in Plochy.dsk one on line.
3. In desktop_index.dsk write an index of the desktop you are curentli using, it must be in Plochy.dsk and it start from 0.
4. Move icons in the folders you created.
5. You can now start the Desktope.py and chose other desktops.

# Desktoper CZ
Program v pythony který řeší problém vitruálních ploch od microsoftu kde není možné mít jiné ikonky na jiných plocháh

# Nastavení
1. Vytvořte šložky a nějak je pojmenujte (v souboru Plochy.dsk jsou nějaké návrhy).
2. Zapište jména těch složek do Plochy.dsk každé na samostatný řádek.
3. V souboru desktop_index.dsk napište index zobrazené plochy (té kterou teď máte), číslovaní začíná od 0.
4. Roztřiďte ikony a nějaké nechte na ploše nebo můžete vytvořit prázdnou plochu kterou poté smažete, ale program podle indexu předpokládá, že na ploše již něco je a bude to přesouvat do té složky.
5. Teď stačí akorát zapnout soubor Desktoper.py a můžete mněnit plochy.
