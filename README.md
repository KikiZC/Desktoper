# Desktoper CZ
Program v pythony který řeší problém vitruálních ploch od microsoftu kde není možné mít jiné ikonky na jiných plocháh.

# Nastavení
1. Vytvořte šložky a nějak je pojmenujte (v souboru Plochy.dsk jsou nějaké návrhy).
2. Zapište jména těch složek do Plochy.dsk každé na samostatný řádek.
3. V souboru desktop_index.dsk napište index zobrazené plochy (té kterou teď máte), číslovaní začíná od 0.
4. Roztřiďte ikony a nějaké nechte na ploše nebo můžete vytvořit prázdnou plochu kterou poté smažete, ale program podle indexu předpokládá, že na ploše již něco je a bude to přesouvat do té složky.
5. Teď stačí akorát zapnout soubor Desktoper.exe a můžete mněnit plochy.

# Desktoper EN
A program in Python that solves the problem of virtual desktops from Microsoft where it is not possible to have other icons on other desktops.

# How to set up
1. Create folders and name them something (there are some suggestions in the Plochy.dsk file).
2. Write the names of those folders in Plochy.dsk, each on a separate line.
3. In the desktop_index.dsk file, write the index of the displayed desktop (the one you have now), numbering starts from 0.
4. Sort the icons and leave some on the desktop, or you can create an empty desktop which you can then delete, but the program assumes that there is already something on the desktop based on the index and will move it to that folder.
5. Now all you have to do is turn on the Desktoper.exe file and you can change desktops.
